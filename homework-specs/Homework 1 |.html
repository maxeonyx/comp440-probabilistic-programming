Homework # 1

Learning Goal

Refresh your understanding of Bayesian inference. Work through the manual exercises whose automation drove, initially, the development of probabilistic programming languages and systems. This homework is long and is designed to rapidly establish the appropriate level of probabilistic reasoning fluency and practical ability required to succeed in the course.

Format

Please write up your answers using LaTeX and generate a .pdf. Include critical code snippets sufficient to judge your authorship of the code and its correctness. Please submit your homework to gradescope using the instructions provided in class. The homework is due at midnight PST on evening of the indicated due date.

Questions

1) (2 points) Show that the Gamma distribution is conjugate to the Poisson distribution.

2) (2 points) Show that the Gibbs transition operator satisfies the detailed balance equation and as such can be interpreted as an MH transition operator that always accepts.

3) (10 points) Write code to compute the probability three ways that it is cloudy given that we observe that the grass is wet using this Bayes net model.

  • By enumerating all possible world states and conditioning by counting which proportion are cloudy given the observed world characteristic, namely, that the grass is wet
  • Using ancestral sampling and rejection
  • Using Gibbs sampling

4) (10 points) Consider the Bayesian linear regression model discussed in the lecture on graphical models

It has joint

p(t,x,σ2,w,α)=∏n=1Np(tn|w,xn,σ2)p(w|α)p(t,x,σ2,w,α)=∏n=1Np(tn|w,xn,σ2)p(w|α)

likelihood

tn|w,xn,σ2∼Normal(wTxn,σ2)tn|w,xn,σ2∼Normal(wTxn,σ2)

and prior

w|α∼∏d=1DNormal(0,α)=Normal(w|0,αId×d)w|α∼∏d=1DNormal(0,α)=Normal(w|0,αId×d)

Show and derive the updates required to

  • perform MH within Gibbs on the blocks ww and t^t^.
  • perform pure Gibbs on both ww and t^t^.
  • produce the analytic form of the posterior predictive.

5) (20 points) Implement a sampler for the LDA model on a corpus consisting of abstracts from NeurIPS in years past. You may wish to start from support code that will be provided on the course website. Use the output to answer questions about the NIPS abstract corpus. The data for this model are in bagofwords_nips.mat, words_nips.mat, and title_nips.mat. When you load these files in the variables DSDS, WSWS, WOWO, and titlestitles will appear in your workspace. The variable WSWS is the entire corpus vectorized into a long row of words encoded as a row vector of integers. Each integer represents a word, WOWO is the dictionary; to look up the word corresponding to an integer use WOiWOi. The variable DSDS is a row vector of the same length as WSWS that indicates from which document each word comes. If the document number is jj then the title of that document can be found using titles[j]titles[j]. Train LDA with 20 topics using an MCMC sampler in the collapsed representation (i.e. with all βkβk’s and θdθd’s integrated out analytically) and use the single sample with the highest joint log likelihood to answer the following questions :

  • What are the top ten most probable words for each of the 20 topics?
  • One can consider the distribution over topics as a low dimensional representation of a document. We can use the dot product between topic distributions for two documents as a similarity metric. What are the ten most similar documents to the first document, “Connectivity Versus Entropy”?
